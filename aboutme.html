Sobre Mim!

Olá meu nome é Enos, tenho 19 anos, durante um bom tempo cursei o ensino técnico em informática no qual pude desenvolver habilidades na área da programação. O curso abrangeu diversas habilidades necessárias como metodologias ágeis, padrões de projeto (design patterns), Banco de Dados (MySQL), diagramas UML e ER, linguagens de programação como Java e C#.

Durante outro período pude cursar Desenvolvimento de Sistemas que me garantiu habilidades para desenvolver em python utilizando micro frameworks como Flask, criando aplicações webs que me garantiram conhecimentos como JavaScript • HTML • CSS.

Agora nesse momento tenho a oportunidade de cursar Sistemas de informções que continuará aprimorando minhas habilidades na programação. Estou em constante aprimoramento e aprendizado, sou uma pessoa de fácil comunicação e tenho ótimas habilidades para desenvolver em grupo, esse sou eu.
